# 링크 분석 및 제안

볼트의 노트 간 연결을 분석하고 누락된 링크를 제안한다.
"/vault-link", "링크 분석", "연결 분석" 등의 키워드에 반응.

## 절차

1. **노트 수집**: Glob으로 볼트 전체 `.md` 파일 수집 (제외: `_templates/`, `.obsidian/`, `.claude/`, `_maintenance/`)
2. **메타데이터 추출**: 각 노트에서 추출:
   - frontmatter: type, tags
   - 제목 (H1)
   - 기존 outgoing `[[wikilink]]`
   - H2 헤딩 키워드
3. **유사도 분석**:
   - 공유 태그가 있지만 링크되지 않은 노트 쌍
   - 제목/헤딩 키워���가 다른 노트 본문에 등장하지만 링크 없는 경우
   - 같은 프로젝트 태그를 가진 노트 간 미연결
4. **제안 출력**: 발견된 연결 기회를 목록으로 출력
   - 형식: `[[note-a]] ↔ [[note-b]]: 공유 태그 #tag, 키워드 일치 "keyword"`
   - 확신도 높은 순으로 정렬
5. **적용**: 사용자 확인 후, 선택된 쌍의 `## 관련 노트` 섹션에 상호 링크 추가

## 규칙

- 분석 대상이 50개 이상이면 subagent에 위임
- 링크 추가 시 양방향 (A→B, B→A) 모두 추가
- 이미 링크된 쌍은 제안에서 제외
