# Building a Second Brain

## 요약

Tiago Forte의 PKM 방법론. 지식을 **CODE**(Capture → Organize → Distill → Express) 흐름으로 처리하고, **PARA**(Projects, Areas, Resources, Archives) 구조로 정리한다.

핵심 통찰: "두 번째 뇌는 외부 저장소가 아니라 **사고의 도구**"다. 단순히 쌓는 것이 아니라 다시 꺼내어 연결하고 표현하는 과정에서 가치를 낸다.

## 인용

> "Your mind is for having ideas, not holding them."
> — David Allen via Forte

## 집현과의 관계

집현전은 CODE 중 특히 **Distill → Express** 단계를 LLM에 위임하는 것이 목표. Capture/Organize는 사람이 하되, 요약·연결·모순 점검은 Claude가 북키핑한다.

PARA는 우리 볼트의 `inbox/ dev/ decisions/ readings/ meetings/ ideas/ artifacts/ projects/` 구조와 유사하지만 더 타입 기반이다.
